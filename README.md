Using Excel for transformation and Power BI for visualisation, the project turns raw health dataset into actionable insights

## Key Insights
* The hospital currently have 47 female patients that are smoking and have a healthy liver
* The BP risk for patients of age 60+ is high
* Patients of age 59 - 68 take an average of 7 cigarrates per day

# Recommendations
* Awareness Campaigns: Launch educational campaigns about the risks of smoking
* Offer routine liver function tests for patients who smoke, even if they currently have healthy liver function
